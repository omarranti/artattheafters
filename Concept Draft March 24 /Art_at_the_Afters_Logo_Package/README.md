# Art at the Afters — Logo Package
## by Stevie Alger

### Concept 3 — Brushstroke "A"
A hand-painted letter "A" in hot pink with an electric green crossbar. 
Elegant, bold, and instantly recognizable at any size.

- `concept3_lockup_dark.svg` — Full horizontal lockup (icon + wordmark + byline)
- `concept3_stacked_dark.svg` — Centered/stacked version for social headers
- `concept3_icon_transparent.svg` — Icon mark only, transparent background
- `concept3_favicon_512.svg` — Square favicon/app icon (512x512, rounded corners)

### Concept 4 — Paint Splash 8-Ball
An 8-ball with "A" inside, exploding out of a hot pink paint splash with green accents.
Louder, more playful, built for social media and merch.

- `concept4_lockup_dark.svg` — Full horizontal lockup (icon + wordmark + byline)
- `concept4_stacked_dark.svg` — Centered/stacked version for social headers
- `concept4_icon_transparent.svg` — Icon mark only, transparent background
- `concept4_favicon_512.svg` — Square favicon/app icon (512x512, rounded corners)
- `concept4_ig_circle.svg` — Instagram profile picture (circular crop, 400x400)

### Shared
- `wordmark_only_dark.svg` — Text-only wordmark with byline and tagline

### Brand Colors
- Afters Black: #1A1A1A
- Hot Pink: #FF3399
- Electric Green: #39FF33
- White: #FFFFFF

### Fonts Used
- Headings/Wordmark: Georgia (serif), Bold
- "at the" / Byline: Arial (sans-serif)
- "by Stevie Alger": Georgia italic

### Usage Notes
- All SVGs are vector and scale infinitely
- Icon marks (transparent bg) work on any background color
- Favicon SVGs include dark background and rounded corners
- For print, export at 300dpi minimum
